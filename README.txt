NAME: EBUBE ABARA
DATE: SEPTEMBER 2016
TASK: Program accepts user location as a pair of coordinates,
      and returns a list of the five closest events, along
      with the cheapest ticket price for each event.

---------------------------------------------------------------------------

1) The program was built in Visual Studio Community 2015 IDE and is 
   a Console application

2) All code was programmed using C#

3) File: viagogo > Program.cs

4) Build program: ctrl + shift + b

5) Run program: ctrl + f5

---------------------------------------------------------------------------

INSTRUCTION ON HOW TO INPUT COORDINATES

1) Enter x coordinate
2) Press ENTER button
3) Enter y coordinate

---------------------------------------------------------------------------

ASSUMPTIONS IN PROGRAM

*) All values are randomly generated

*) Event unique identifier goes from 1 to 30

*) Ticket prices range from $0.01 to $99.99
